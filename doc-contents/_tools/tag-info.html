--- layout: tag permalink: /tag-info title: Tag tool ---
Click on a tag to see the pages having that tag. Click on "Read" button to navigate to the page.
Click on "Save" button to save the page for later read. If the button is disabled, the page is already in the saved pages list and the "Remove" button should be enabled.
Click on "Remove" button to remove the page from the list of saved pages. If the button is disabled, the page is not in the saved pages list and the "Save" button should be enabled.
